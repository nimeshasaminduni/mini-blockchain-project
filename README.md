# Mini Blockchain Project

A simple blockchain project built using Python.

## Features

- Block creation
- SHA-256 hashing
- Mining system
- Blockchain validation
- Merkle root generation
- Wallet transactions
- Dynamic difficulty
- Simple Proof of Stake demo

## Technologies Used

- Python
- Hashlib
- JSON
- Datetime

## How to Run

### 1. Clone Repository

```bash
git clone https://github.com/nimeshasaminduni/mini-blockchain-project.git
```

### 2. Open Project Folder

```bash
cd mini-blockchain-project
```

### 3. Run Project

```bash
python main.py
```
